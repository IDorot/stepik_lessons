Тестовый сценарий 1.1.1. Проверка корректного изменение кол-ва позиций и расчёт их стоимости.

Предусловия:
- В корзину добавлена хотя бы одна позиция.
- Значение в графе количество является целым числом больше или равно 0.

Шаги:
- Шаг 1, изменить количество товарной позиции. Нажать обновить.  Ожидаемый результат - количество позиций изменилось корректно, стоимость заказа изменилось согласно новому количеству. 
- Шаг 2, удалить любую позицию с помощью соответствующей кнопки. Ожидаемый результат - позиция удалена. Стоимость заказана перерасчитана корректно.
- Шаг 3, удалить все позиции из корзины. Ожидаемый результат - корзина пуста. Появление кнопки для перехода на главную.

Тестовый сценарий 1.2.1. Ввод промокода.

Предусловия:
- В корзину добавлена хотя бы одна позиция.
- Значение в графе количество является целым числом больше или равно 0.

Шаги:
- Шаг 1, нажать кнопку "У меня есть код скидки". Ожидаемый результат - появление поля ввода для промокода.
- Шаг 2, ввести промокод. Ожидаемый результат - перерасчёт итоговой суммы заказа со скидкой. 




Тестовый сценарий 2.1.1. Добавление товаров в корзину. Проверка отображения суммы заказа и корзины. 

Предусловия:
- Товары имеют имеют стоимость больше 0.


Шаги:
- Шаг 1, выбрать рандомный товар в наличии.
- Шаг 2, прожать кнопку добавить товар. Ожидаемый результат - изменение суммы в графе "Всего в корзине".
- Шаг 3, попытаться прожать кнопку покупки недоступного для заказа товара. Ожидаемый результат - кнопка не прожимается.
- Шаг 4, открыть выпадающий список напротив кнопки с корзиной. Ожидаемый результат - открывается предпросмотр корзины.

 
 Тестовый сценарий 2.2.1. Проверка перевода всех текстовых элементов. 
 
Предусловия:
-Наименования товаров (книг) неизменно.
-Изначальный язык - русский.

Шаги:
- Шаг 1, открыть выпадающий список "Русский" и выбрать любой другой язык.
- Шаг 2, нажать выполнить.  Ожидаемый результат - текстовые эелементы, кроме наименований товаров, переведены на соответствующий язык. Включая валюту.
